## Key Concepts

### Home Purchase Timeline
1. **Pre-qualification/Pre-approval** - Get financing approval estimate
2. **Property Search** - Find property with help of realtor
3. **Make Offer** - Submit purchase agreement
4. **Appraisal** - Lender orders property valuation
5. **Underwriting** - DU system analyzes application
6. **Conditions** - Satisfy additional documentation requests
7. **Clear to Close** - All conditions met, closing scheduled
8. **Closing** - Sign documents, get keys, move in

### Mortgage Market Players

- **Fannie Mae & Freddie Mac** - Buy mortgages from lenders (secondary market)
- **Ginnie Mae** - Securitizes government-backed loans
- **Lenders** - Banks, mortgage companies, credit unions
- **Servicers** - Collect payments after loan closes
- **Appraisers** - Determine property value
- **Realtors** - Help buy/sell properties
- **Title Companies** - Verify property ownership
- **Underwriters** - Verify borrower qualifies for loan

### Common Loan Programs

- **Conventional** - Non-government loans (10-20% down)
- **FHA** - First-time buyer friendly (3.5% down)
- **VA** - For veterans (0% down, no PMI)
- **USDA** - For rural properties (0% down)

## USA Mortgage Facts

- **Typical down payment:** 3-20%
- **Typical interest rate:** 5-8% (varies by market)
- **Typical loan term:** 30 years (15-year also popular)
- **PMI threshold:** Below 20% down requires PMI
- **Loan limit (2024):** $766,550 (Fannie Mae conventional)
- **Average processing time:** 30-45 days
- **Closing costs:** Typically 2-5% of loan amount
